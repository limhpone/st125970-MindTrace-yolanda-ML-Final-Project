Research Question — Section 1.3
How do traditional ML models (SVM, XGBoost) compare with deep learning models (CNN, BiLSTM) in multi-class emotion classification from social media text, when evaluated under a unified preprocessing pipeline, standardised dataset partitioning, and consistent evaluation metrics?
Research Gap — Section 1.2
Comparative analyses across studies frequently differ in dataset selection, preprocessing strategies, feature engineering, and evaluation metrics — making observed performance differences attributable to experimental inconsistencies rather than intrinsic model capabilities. This study fixes all conditions and varies only the model family.
Dataset — Section 3.1
Kaggle Emotions dataset: 416,809 labelled Twitter samples across 6 classes (anger, joy, sadness, fear, surprise, love). Justified by: large sample size for statistical reliability, real-world informal language for ecological validity, and six-class taxonomy for fine-grained emotion analysis beyond binary sentiment polarity.
NLP Pipeline — Section 3.4
  • 01Lowercasing — normalise case
  • 02Whitespace stripping — collapse multiple spaces
  • 03URL removal — strip http/www links
  • 04Emoji removal — via emoji library
  • 05Special character removal — keep a–z and spaces
  • 06Chat word expansion — u→you, lol→laugh out loud, etc.
  • 07Stopword removal — negation words preserved (not, never, very, extremely…)
  • 08Lemmatisation — NLTK WordNetLemmatizer
Four Models — Section 4
BiLSTMCaptures long-range bidirectional contextual dependencies; highest test accuracy 93.9%
CNNExtracts local n-gram features via convolution and pooling; test accuracy 92.5%
SVMMargin-based classifier on TF-IDF sparse features; strong classical baseline 91.8%
XGBoostRegularised gradient-boosted ensemble on TF-IDF; non-linear baseline 90.8%
Key EDA Findings — Section 3.2
  • ↳Class imbalance 9.5:1 (Joy:Surprise) → macro-F1 evaluation required
  • ↳Text length right-skewed; median 85 chars, range 11–512
  • ↳text_length ↔ word_count correlation r = 0.984 (structurally redundant)
  • ↳Label ↔ length correlation r < 0.03 → semantic content is the primary signal
  • ↳Love class longest (mean 104.7 chars); Sadness shortest (mean 93.1 chars)
Contribution — Section 6.1
Unified experimental framework — identical preprocessing, splits, and metrics for all four models
Class-wise confusion matrix analysis with focus on semantically similar pairs (Fear/Surprise, Sadness/Love)

End-to-end REST API deployment pipeline with Flask + Docker containerisation
