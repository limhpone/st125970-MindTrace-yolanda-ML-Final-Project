\section{Contribution, Conclusion, and Future Work}
\label{sec:conclusion}

\subsection{Contribution}

This study contributes a reproducible experimental benchmark for text-based emotion classification by evaluating SVM, XGBoost, CNN, and BiLSTM under identical preprocessing, stratified splits, and evaluation metrics on a 416,809-sample Twitter corpus. Unlike prior work that varies these conditions across studies, this research isolates intrinsic model differences. The complete pipeline --- preprocessing scripts, training code, and performance comparison --- is publicly available for future researchers to extend.

\subsection{Conclusion}

BiLSTM achieved the highest test accuracy (93.93\%, macro-F1 0.9391), outperforming CNN (92.54\%), SVM (91.80\%), and XGBoost (90.79\%), confirming that bidirectional contextual encoding provides a measurable advantage for multi-class emotion classification. Confusion matrix analysis revealed that Joy$\leftrightarrow$Love and Fear$\rightarrow$Surprise were the dominant misclassification pairs across all architectures, validating that semantic overlap --- not model capacity --- is the primary classification bottleneck.

Despite achieving the highest accuracy, BiLSTM was not selected for deployment due to ${\sim}23\times$ higher inference latency. SVM was deployed as a Flask-based REST API containerised with Docker, demonstrating that a classical model with TF-IDF features can deliver reliable real-time predictions at minimal cost.

\subsection{Future Work}

Future work will extend the framework in four directions:
\begin{enumerate}[nosep]
    \item Fine-tuning transformer-based models (BERT, RoBERTa) on the same dataset to compare against the baselines established here.
    \item Cross-domain evaluation on non-Twitter corpora (Reddit, product reviews) to assess generalisation.
    \item Scaling the deployed API with a real-time emotion monitoring dashboard.
    \item Explainability analysis using SHAP and LIME to identify token-level features discriminating between semantically similar emotion classes.
\end{enumerate}
